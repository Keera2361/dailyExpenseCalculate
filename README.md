# Daily Expenses Calculator

A simple application to track and calculate daily expenses.

## Features

* Add daily expenses
* View total expenses
* Manage expense records
* Simple and easy interface

## Technologies Used

* Node.js
* Express.js
* MongoDB
* HTML / CSS / JavaScript

## Installation

1. Clone the repository

```
git clone https://github.com/Keera2361/dailyExpenseCalculate.git
```

2. Navigate to the project folder

```
cd dailyExpenseCalculate
```

3. Install dependencies

```
npm install
```

4. Start the server

```
npm start
```
